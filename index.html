Working with Lists in HTML

HTML provides two types of lists: unordered lists and ordered lists.

Unordered List

An unordered list is a collection of items that have no particular order. The list items are typically displayed with bullets.

Example:

<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Orange</li>
</ul>

Result:

  • Apple
  • Banana
  • Orange

Ordered List

An ordered list is a collection of items where the order matters. Each item is numbered by default.

Example:

<ol>
  <li>First</li>
  <li>Second</li>
  <li>Third</li>
</ol>

Result:

  1. First
  2. Second
  3. Third

Nested Lists

You can also create nested lists, meaning a list inside another list. Here's an example of a nested unordered list:

<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
  <li>Vegetables
    <ul>
      <li>Carrot</li>
      <li>Broccoli</li>
    </ul>
  </li>
</ul>

Result:

  • Fruits
    • Apple
    • Banana
  • Vegetables
    • Carrot
    • Broccoli

List with Custom Bullet Points

You can customize the bullets in unordered lists using the list-style-type CSS property.

ul {
    list-style-type: square;
}

Example:

  • Square Bullet 1
  • Square Bullet 2
  • Square Bullet 3

Conclusion

HTML lists are a great way to present items in a structured format, either in an unordered way (with bullets) or in an ordered sequence (with numbers). You can also nest lists to create more complex structures.
